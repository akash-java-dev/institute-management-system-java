# Institute Management System (Java)

## Description
A console-based Java application to manage institute operations such as students, faculty, courses, and batches.

## Features
- Student Management
- Faculty Management
- Course Handling
- Batch Management

## Tech Stack
- Java
- OOP Concepts
- Collections Framework

## Project Structure
- model → Data classes (Student, Faculty, Course, Batch)
- service → Interfaces
- serviceimpl → Implementations
- client → Main execution

## How to Run
1. Clone the repository
2. Open in Eclipse/IntelliJ
3. Run `Test.java`
